## About

This repository contains Python and AI-powered automation projects built to 
solve real GTM and RevOps business problems. Each project is scoped around 
the tools, workflows, and challenges that modern GTM Engineering and Revenue 
Operations roles demand — from CRM data operations to pipeline intelligence 
and outreach automation.

Built to demonstrate hands-on capability across the RevOps and GTM stack 
using Python, Claude Code, and HubSpot integrations.
